## mkdocs
- ftp_push
- mkdocs_build
- push_mkdocs_repository
  
### 使い方
- 前提条件
  - Pythonインストール済み
  - pipインストール済み
  - VScodeインストール済み
  
- mkdocs インストール
```
    pip install mkdocs-material
```
- サーバー起動
```
    mkdocs serve
```
- サーバー終了
```
    Ctrl + C
```
